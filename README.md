# Flappy Bird on Aptos
"Flappy Bird on Aptos" is a unique integration of the classic Flappy Bird game with the Aptos blockchain, where users can enjoy playing while minting bird NFTs. The game features a competitive leaderboard, and the top players receive exclusive NFTs as rewards. Leveraging Aptos's high transactions per second (TPS) capability ensures smooth gameplay, even with a large number of players simultaneously engaging in the game. **Project is still in progress and Users might face some unintended errors.**


This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).
